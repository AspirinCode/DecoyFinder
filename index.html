DecoyFinder by ssorgatem

Tool to find decoy molecule sets for given active ligand sets

DecoyFinder is a graphical tool which helps finding sets of decoy molecules for a given group of active ligands. It does so by finding molecules which have a similar number of rotational bonds, hydrogen bond acceptors, hydrogen bond donors, logP value and molecular weight, but are chemically different, which is defined by a maximum Tanimoto value threshold between active ligand and decoy molecule MACCS fingerprints. Optionally, a maximum Tanimoto value threshold can be set between decoys in order assure chemical diversity in the decoy set.

Dependencies

Python +2.6, OpenBabel >= 2.3.0 with python bindings, either PySide or PyQt4 Qt bindings for python.

Install

* Debian/Ubuntu and derivatives: open and install the .deb packages with a graphical package manager or with dpkg from the command line. * Others: No installation is required provided you have installed in your system OpenBabel >= 2.3 with python bindings, Python >= 2.6 and python Qt bindings (PyQt4 or PySide, depending on the version of DecoyFinder you downloaded). Just unpack the tarball and execute decoy_finder.py
