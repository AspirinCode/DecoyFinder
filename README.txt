Release Notes v1.0

    * Due to a bug in OpenBabel <= 2.3, files compressed with gzip or bz2 are not supported on Windows.


Dependencies:
- Core:
    - Python >= 2.5,  < 3
    - argparse python module (if using python < 2.7)
    - OpenBabel >= 2.3
    - pybel

- GUI:
    - PySide >= 1.0 (or PyQt4 >= 4.6 for PyQt4 version)
    - Qt >= 4.6
